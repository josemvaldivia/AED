Detailed Description

template<typename D>
class TFor< D >

<

La clase TFor depende de un D, que es el distribuidor de carga, a este se carga la funcion con la respectiva clase STL, el TFor lanzara los threads segun lo que se indique, por defecto será el hardware_concurrency().
